# SowetoMarket

## Project Overview

SowetoMarket is a Customer-to-Customer (C2C) e-commerce platform developed for community members to buy and sell products online in a secure and user-friendly environment.

## Live Website

https://sowetomarket.fwh.is

## Features

### User Features

* User Registration
* User Login and Logout
* Account Management
* Product Search
* Category Filtering
* Product Status (Available, Reserved, Sold)
* Product Ratings
* Messaging System
* Seller Inbox
* Reply to Messages
* Product Listings Management

### Seller Features

* Post Products
* Edit Products
* Delete Products
* View Listings
* Receive Ratings
* Receive Messages

### Admin Features

* Admin Dashboard
* Manage Users
* Delete Users
* Manage Products
* Delete Products
* View Platform Statistics

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* PHP
* MySQL
* JavaScript

## Database

The project uses MySQL and includes:

* Users
* Products
* Categories
* Messages
* Ratings

Database export file:

sowetomarket.sql

## Installation

1. Import the database using phpMyAdmin.
2. Configure database settings in `includes/db.php`.
3. Upload files to a PHP hosting server.
4. Run the website through the hosting URL.
